# 云栖按键工具 (v1.0)
Windows 轻量级按键连发工具：自动顺序交替连发/单次点击，支持快捷键启停，无需复杂操作，可打包成独立EXE文件直接运行，提升操作效率。

# 工具功能
- **顺序交替连发**：支持多按键按列表顺序循环交替点击，自定义按键触发间隔，无杂乱并发点击。
- **双模式切换**：「开关模式」持续循环连发至手动停止；「单次模式」执行一轮勾选按键后自动停止，按需选择。
- **快捷启停方式**：支持F1~F12全局快捷键一键启停，同时保留界面按钮手动启停，双方式兼容使用。
- **便捷按键管理**：点击「添加按键」后，按下任意键盘按键即可自动捕获添加，选中列表行点击删除即可移除，操作简单。
- **轻量无依赖**：适配Windows全版本，原生系统支持，无广告无捆绑，打包后为单文件，开箱即用。

# 安装步骤
### 方法：源码运行（本地部署）
1. 下载项目源码，解压至本地任意目录（**路径请勿包含中文/空格/特殊字符**）。
2. 打开命令提示符，进入源码目录，执行命令安装依赖插件：
    ```bash
    pip install PyQt5 pynput
    ```
3. 依赖安装完成后，执行启动命令即可运行工具：
    ```bash
    python main.py
    ```

# 使用说明
### 基础操作
1. **添加/删除按键**：点击「添加按键」，按钮置灰后按下需要的键盘按键，自动添加至列表并默认勾选启用；选中列表目标行，点击「删除按键」即可移除对应按键。
2. **模式与参数设置**：下拉框选择「开关模式/单次模式」，表格内可自定义每个按键的触发间隔（单位：毫秒，默认10ms）。
3. **启停方式**：点击界面「启动」按钮开始运行，「停止」按钮终止运行；也可在右侧下拉框选择F1~F12快捷键，按下对应按键即可一键启停。

### 打包成独立EXE文件
如需脱离Python环境运行，可在源码目录执行打包命令，生成单文件EXE，可随意复制使用：
1. 先安装打包工具：
    ```bash
    pip install pyinstaller
    ```
2. 执行打包命令：
    ```bash
    pyinstaller -w -F -n 云栖按键工具 main.py
    ```
3. 打包完成后，在源码目录的`dist`文件夹内，即可找到生成的EXE文件，双击即可运行。

### 模块切换说明
- 切换至「宏按键」模块时，工具将自动停止当前所有运行任务、保存已配置的按键信息，重置为初始未运行状态。
- 切回「按键连发」模块时，将自动恢复之前保存的按键配置与参数，无需重新设置。

# 常见问题
1. **工具启动闪退/运行报错**：
    - 检查源码目录路径是否包含中文、空格或特殊字符；确认本地已安装Python3.7及以上版本，依赖库安装完整。
2. **按键连发无效果/失效**：
    - 尝试以「管理员身份」运行工具/EXE文件，避免系统权限限制导致按键模拟失效。
3. **打包后EXE文件闪退**：
    - 重新执行打包命令并去掉`-w`参数，运行生成的EXE查看控制台报错信息；确保打包时源码目录无中文路径。
4. **快捷键启停不生效**：
    - 确认所选快捷键（F1~F12）未被其他软件占用，切换未被占用的快捷键即可正常使用。
